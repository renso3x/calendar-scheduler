# Calendar Scheduler Stats

## App Info

> React, Redux Saga, Express, MySQL app that uses the Calendar Sceduler API to display calendar

## Quick Start

```bash
# Install dependencies (server & client)
yarn install
cd client && yarn install

# Run server & client (:3000 & :4000)
yarn run dev

```
